Quick setup (2 minutes)

  1. Install the VPN app for Windows (from the official site or Microsoft Store).
  2. Sign in → set Protocol: WireGuard/NordLynx (first choice).
  3. Enable Kill switch and DNS leak protection.
  4. Connect to a nearby UK server (London/Manchester/Birmingham).

Need Mac or mobile? See macOS, Android, iOS.

Best settings for UK networks

  • Protocol: WireGuard for speed; on restrictive networks use OpenVPN TCP/443.
  • Auto-connect: on startup and on untrusted Wi-Fi.
  • Split tunneling: keep work/banking outside the tunnel only if needed.
  • Private DNS: use provider DNS inside the tunnel.
  • Threat blocking: optional to reduce ads/trackers.

Switch to TCP/443 if hotel/office Wi-Fi interferes with UDP traffic.

Privacy & leak checks

  1. Connect the VPN → run an Extended test from our leak guide.
  2. You should not see your ISP’s DNS or your real public IP.
  3. Flush Windows DNS if needed: open cmd → ipconfig /flushdns.
  4. Test the kill switch by temporarily quitting the app — traffic should stop.

Streaming (Netflix UK, BBC iPlayer)

  1. Pick a low-load UK server; if you see a proxy error, rotate UK city.
  2. Relaunch the app after each change; clear app cache if needed.
  3. On restrictive networks use OpenVPN TCP/443.

Guides: Netflix UK • BBC iPlayer.

Speed optimisation

  • Prefer WireGuard; if packet loss, try TCP/443.
  • Choose the closest UK server; avoid overloaded nodes.
  • Pause background syncs/downloads during streaming.
  • Use Ethernet for 4K streaming and large downloads.
