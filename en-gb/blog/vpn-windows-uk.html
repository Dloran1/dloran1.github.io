Quick setup (2 minutes)

  1. Install NordVPN or Surfshark for Windows.
  2. Open the VPN app → set protocol to WireGuard/NordLynx. Turn on the in-app Kill Switch.
  3. Connect to a low-load UK server (London/Manchester/Birmingham). Confirm your public IP is the VPN’s.

New to VPNs? Start with What is a VPN? and our Best VPN Settings (UK).

WireGuard vs OpenVPN on Windows

  • WireGuard/NordLynx — fastest and CPU-light; excellent for UK broadband and 5G tethering.
  • OpenVPN UDP — stable fallback; good when specific enterprise proxies interfere with WireGuard.
  • OpenVPN TCP/443 — resembles HTTPS, useful for restrictive hotel/office Wi-Fi (expect slightly lower peak speeds).

Best app & OS settings (UK)

1) In the VPN app

  • Protocol: WireGuard/NordLynx (default). Keep OpenVPN TCP/443 as the “break glass” option on strict networks.
  • Kill Switch: Enable. Prefer “system-wide” mode (blocks traffic if the tunnel drops).
  • Use provider DNS: On. Avoid ISP DNS to reduce proxy/unblocker detections.
  • Auto-connect on boot: On if the machine is always online.
  • Obfuscation: Off by default; enable when the LAN blocks VPN protocols.

2) In Windows

  • Metered connection: If you tether via 5G, set the Wi-Fi as metered to limit background chatter.
  • Firewall: Keep Windows Defender Firewall on; allow the VPN app during install so it can create rules.
  • Startup apps: Disable heavy auto-launchers that compete for bandwidth at boot.
  • Power plan (laptops): Use “Balanced”; avoid sleep during large downloads to keep the tunnel alive.

3) Server choice (UK)

  • Pick low-load UK servers close to you. Rotate endpoints at peak time (evenings/weekends).
  • For travel, select the required country and relaunch apps (Netflix/iPlayer) after switching.

Split tunneling on Windows

Split tunneling is useful when some apps dislike VPN egress:

  • Exclude from VPN: UK banking, corporate VPN/SSO, printer/scanner utilities, local game launchers.
  • Keep inside VPN: browsers, streaming apps, P2P clients, messaging.

For whole-home coverage (Smart TVs/consoles), consider a router VPN with policy routing: VPN Router Setup (UK), VPN on a Router.

Streaming (Netflix, BBC iPlayer, more)

  • Netflix UK: If you see a proxy/unblocker message, rotate UK servers → close the Netflix app/browser completely → relaunch. On strict networks switch to TCP/443. Full guide: Netflix UK with a VPN.
  • BBC iPlayer: Use recommended UK endpoints, relaunch the app after each change. Guide: BBC iPlayer with a VPN.
  • Smart TVs: Install the VPN app on Windows and share the connection as a hotspot, or run the VPN on your router. See Smart TV (UK).

Performance & speed tips

  • Prefer Ethernet over Wi-Fi for desktops; on Wi-Fi use 5 GHz.
  • Use WireGuard for maximum throughput. If speeds dip, rotate to a less busy UK endpoint.
  • Pause heavy cloud syncs (OneDrive/Dropbox) while streaming/gaming.
  • For P2P, limit global connections if your router is budget-class; see Torrenting (UK).

IP/DNS/WebRTC leaks on Windows

Check after installation and major updates. If a leak test shows ISP DNS or your real IP:

  • Reconnect the VPN with “Use provider DNS” enabled.
  • Clear OS DNS cache: open cmd as admin → ipconfig /flushdns.
  • Restart the browser; in Chromium-based browsers consider disabling WebRTC IP leak via extensions/settings.
