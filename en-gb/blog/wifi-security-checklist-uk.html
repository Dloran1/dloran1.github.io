Quick checklist

  1. Enable WPA3 (or WPA2-AES if WPA3 isnâ€™t supported).
  2. Use a strong Wi-Fi password and a different strong router-admin password.
  3. Change SSID, avoid brand/model hints.
  4. Keep router firmware up to date (auto-updates if available).
  5. Disable WPS and legacy ciphers (WEP/TKIP).
  6. Turn on the router firewall; block remote admin from WAN.
  7. Segment your network: guests / IoT / your devices (separate SSIDs/VLANs).
  8. Use privacy-friendly DNS with DoH/DoT and malware filtering.
  9. On mobiles, enable auto-VPN on untrusted Wi-Fi.
  10. Disable legacy services (UPnP, Telnet). Open ports only if required.
  11. Enable 2FA on key accounts (email, banking, cloud).
  12. Audit connected devices regularly; remove unknown ones.
  13. Schedule Wi-Fi off at night for less exposure.
  14. Keep system firewalls and auto-updates on.
  15. Restrict app permissions (location, mic, photos) on phones.
  16. For remote work use modern protocols (WireGuard / IKEv2).
  17. Use a reputable VPN for sensitive tasks.

FAQ

Does hiding the SSID improve security?
Only marginally. WPA3 and strong passwords are what really matter.
WPA2 or WPA3?
Prefer WPA3; if unsupported, use WPA2-AES (not TKIP).
Are ISP DNS servers fine?
Use privacy-friendly DNS with DoH/DoT and malware filtering; set it on the router.
